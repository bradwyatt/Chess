# Chess
Chess created using pygame but without using any chess-related libraries

## Prerequisites
Python 3.8.8

```
numpy==1.20.2
pygame==2.0.1
pandas==1.2.3
PySimpleGUI==4.49.0
```

## Features
Play against CPU\
Game Properties (Manually enter the player names, ELO ratings, etc)\
Save PGN\
Load PGN\
Flip Board\
Reset Board\
Clear Board\
Save Position\
Load Position
